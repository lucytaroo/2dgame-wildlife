## Play the Game

How to Play WildLife (2D Game)

1. Click the download link given at the bottom of the instruction

2. Unzip the downloaded folder
   - Right-click the downloaded .zip file → Extract All…
   - Choose a folder to extract the files (e.g., C:\Users\YourName\Games\SultanQuest2D).
   
6. Run the game
   - Open the extracted folder.
   - Double-click the file:
     - Sultan Quest Lost.exe

Enjoy playing the game! 🎮
